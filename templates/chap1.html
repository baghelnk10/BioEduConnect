The Fundamental Unit of Life: Cell

Learning Outcomes

This section would outline the goals of the learning module and what learners should be able to achieve by the end. It could also include a brief overview of the topics covered.

Learn More
×
  1. You should be able to describe the discovery of cells.
  2. You should be able to explain cell theory.
  3. You should be able to describe the structural organization of a cell.
  4. You should be able to list the various components of a cell and state their functions.
  5. You should be able to differentiate between prokaryotic and eukaryotic cells.
  6. You should be able to draw and label plant and animal cells.
  7. You should be able to have a basic understanding of chromosomes, DNA, and types of cell division.
